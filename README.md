# chapel-clojure-app

## Installation

    $ git clone https://github.com/elect000/chapel-clojure-apps ccapps 
    $ cd ccapps
1. from leiningen

        $ lein run
2. from jar

        $ lein uberjar
        $ cd target 
        $ java -jar chapel-clojure-app-0.1.0-standalone.jar 
## Usage

Click : move
Up/Down arrow-key : zoom in/out

    $ lein run
